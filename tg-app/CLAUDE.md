# CLAUDE.md — Документация проекта tg-app

> Telegram Mini App: каталог услуг бьюти-мастера с записью в 3 шага.

---

## Структура файлов

```
tg-app/
├── index.html          ← точка входа, HTML-разметка всех 5 экранов
```

## Как это работает

### Экраны и переходы

Все 5 экранов всегда в DOM — переключение через CSS-классы:

| Класс           | Значение                              |
|----------------|---------------------------------------|
| `active`        | Экран виден                           |
| `slide-in-right`| Анимация входа справа (вперёд)        |
| `slide-in-left` | Анимация входа слева (назад)          |
| `fade-in`       | Фейд (для экрана успеха)              |

**Навигационный стек экранов:**
```
catalog → service → datetime → confirm → success
```
Назад работает в обратную сторону. С `success` можно только на `catalog` (reset).

### Кнопки Telegram

| Кнопка          | Экран                | Действие                    |
|----------------|----------------------|-----------------------------|
| BackButton      | service              | → catalog                   |
| BackButton      | datetime             | → service                   |
| BackButton      | confirm              | → datetime                  |
| BackButton      | success              | **скрыта** (нельзя назад)   |
| MainButton      | catalog              | **скрыта**                  |
| MainButton      | service              | «Записаться» → datetime     |
| MainButton      | datetime             | «Продолжить» (disabled пока не выбран слот) → confirm |
| MainButton      | confirm              | «Подтвердить запись» → success |
| MainButton      | success              | «Вернуться в каталог» → catalog |

## Где менять данные

### Мастер (имя, рейтинг, рабочие дни)
`js/data.js` → объект `MASTER`:
```js
const MASTER = {
  name:         'Студия Ани',      // название студии
  masterName:   'Анна Смирнова',   // имя мастера (в экране успеха)
  rating:       4.9,
  reviewsCount: 127,
  workDays:     [1,2,3,4,5,6],     // 0=Вс ... 6=Сб (0 = выходной)
  workHours:    { start: 9, end: 20 },
};
```

### Категории фильтров
`js/data.js` → массив `CATEGORIES`:
```js
{ id: 'nails', emoji: '💅', label: 'Ногти' },
```
`id` должен совпадать с `category` в услугах.

### Услуги (цены, описания, длительность)
`js/data.js` → массив `SERVICES`. Каждая услуга:
```js
{
  id:          1,
  category:    'nails',             // id из CATEGORIES
  name:        'Маникюр + гель-лак',
  short:       'Покрытие гелем на 3–4 недели',  // в будущем для карточки
  description: 'Полное описание...',
  duration:    75,                  // минуты
  price:       2500,                // рубли (без ₽)
  emoji:       '💅',                // большой emoji на превью
  gradient:    'linear-gradient(...)', // фон превью-карточки
  photos:      ['💅', '✨', '🌸'],  // emoji для слайдов галереи (3 штуки)
  reviews:     [
    { author: 'Катя', text: '...', date: 'вчера', rating: 5 },
  ],
}
```

### Слоты времени
`js/data.js` → объект `BASE_SLOTS`:
```js
const BASE_SLOTS = {
  morning:   ['09:00', '10:00', '11:00'],
  afternoon: ['13:00', '14:00', '15:00', '16:00'],
  evening:   ['18:00', '19:00', '20:00'],
};
```
Занятые слоты генерируются детерминированно по хешу даты (~20% занято).
В реальном проекте — заменить на API-запрос к бэкенду.

## Ключевые решения

| Задача | Решение |
|---|---|
| Тёмная тема | `var(--tg-theme-*)` — меняется автоматически при смене темы Telegram |
| Браузерный просмотр | Мок-объект `createTgMock()` в `app.js` — имитирует MainButton/BackButton |
| Телефон между сессиями | `tg.CloudStorage.setItem('phone', ...)` — сохраняется после первой записи |
| Имя пользователя | `tg.initDataUnsafe.user.first_name` — берётся из Telegram |
| Haptic feedback | `tg.HapticFeedback.impactOccurred('light'/'medium')` на каждом тапе |
| Закрытие формы | `tg.enableClosingConfirmation()` на экране 4 — защита от случайного свайпа |

## Подключение к боту

1. Создать бота через [@BotFather](https://t.me/BotFather)
2. Задеплоить `tg-app/` на HTTPS-хостинг (Vercel, Netlify и т.д.)
3. В BotFather: `/newapp` → ввести URL задеплоенного `index.html`
4. Добавить кнопку в меню бота: `/setmenubutton`

**Важно:** Telegram Mini App работает только по HTTPS. При локальной разработке используй `ngrok` или `localtunnel`.
